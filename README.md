# Aegis — SaaS Customer Churn Prediction (Live Demo)

This Hugging Face Space runs the **full stack**: a FastAPI prediction API plus
the Streamlit dashboard, in a single always-on container.

- **Source code, architecture & benchmarks:** https://github.com/Lucas-Maingi/aegis-churn-analytics
- **What it does:** predicts a customer's churn probability and returns
  SHAP-based, plain-English reasons so retention teams know who to save and why.

> This is the deployment branch. The README with full documentation, honest
> metrics, and limitations lives on the [`main` branch on GitHub](https://github.com/Lucas-Maingi/aegis-churn-analytics).
